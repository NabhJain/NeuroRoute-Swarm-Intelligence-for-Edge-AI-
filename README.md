# NeuroRoute: Swarm Intelligence for Edge AI

AI framework for **autonomous fleet routing optimization** using **Swarm Intelligence + Federated Learning**.

This project explores how decentralized AI systems can optimize **vehicle routing, eco-driving behavior, and energy consumption** in connected fleets.

---

## Project Overview

NeuroRoute proposes an intelligent routing framework where edge devices (vehicles) collaboratively learn optimal routes without sharing raw data.

The system combines:

* Swarm Intelligence for collective routing decisions
* Federated Learning for distributed model training
* Edge AI for real-time inference inside vehicles

---

## Key Contributions

• Designed a **hybrid Swarm + Federated Learning architecture** for vehicle fleets
• Processed **1,000+ simulated driving scenarios**
• Trained predictive models for **eco-driving optimization**
• Reduced simulated **carbon emissions and battery consumption**
• Enabled **vehicle-to-vehicle cooperative routing**

* Python

---

## System Architecture

The framework operates in a distributed manner:

1. Each vehicle acts as an **Edge AI node**
2. Vehicles train **local driving optimization models**
3. Models are aggregated using **Federated Learning**
4. Swarm Intelligence coordinates routing decisions across the fleet

---

## Results

The NeuroRoute system demonstrated:

* Improved eco-driving scores
* Reduced simulated fleet-wide battery consumption
* Lower carbon emission estimates
* Efficient decentralized learning across vehicles
